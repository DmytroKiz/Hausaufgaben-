Перелік основних команд

  • git config --global user.name blabla - Імя користувача
  • git config --global user.email blabal@bla - імя користувача
  • git init - Створення репозиторію
  • git status - Показує поточний стан проекту (зміни, доповнення тощо)
  • git add - Команда що добавляє зміни в розділ проіндексованих фалів
  • git commit - Використовується для збереження змін
  • git loga - Перегляд історії комітів
  • git checkout - Переключення між гілками
  • git brancha - Команда створює нові гілки
  • git push - Зєднання з відділаним репозиторієм
  • git pull - Оновлює віддалені посилання за допомогою локальних посилань, одночасно надсилаючи об’єкти, необхідні для завершення заданих посилань.а
  • git merge - Команда обєднює гілки в одну
  • git clone- Команда створює копію репозиторію

Advanced

  • git revert - Спосіб відміни операції без попередньої встрати історії комітів
  • git reset - видалення історії та ст оврення
  • git rebase - один із способів зєднати (обєднати зміни) що були зроблені в одній гілці з іншою шілкою
  • git cherry-pick - Команда що дозволяє вибирати довільні коміти Git за посиланням і додавати їх
  • git stash - зберігає непідтверджені зміни в окреме сховище
